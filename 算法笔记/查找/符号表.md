# 符号表

> 符号表是一种存储键值对的数据结构，支持两种操作：插入（put），即将一组新的键值对存入表中；查找（get），即根据给定的键得到相应的值

### 约定

- [ ] 每一个键只能对应着一个值（表中不允许存在重复的键）

- [ ] 当用例代码向表中存入的键值对和表中已有的键冲突时，新的值会代替旧的值
- [ ] 键不能为空。和Java中的许多其他机制一样，使用空键会产生一个运行时异常（调用compareTo方法时）
- [ ] 当键不存在时`get()`方法会返回空
